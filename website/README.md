# Antyx-OS Website

Die Seite ist statisch und benötigt keinen Server mit Datenbank.

Lokal öffnen:

```bash
cd website
python3 -m http.server 8080
```

Danach im Browser `http://localhost:8080` öffnen.

Später kann der Ordner über GitHub Pages, Cloudflare Pages oder einen normalen Webserver veröffentlicht werden. Der Download-Button wird erst aktiviert, wenn eine getestete ISO existiert.
